Projects

Machine Learning for Pipeline Failure Prediction and Risk Mitigation

This project focuses on using advanced machine learning techniques to help a pipeline repairs company accurately predict when pipeline failures are likely to occur. By establishing a risk threshold, a high danger of failure can be indicated when the risk level reaches 80%, prompting immediate action to prevent accidents or further damage.

SparkETL- Failure Detection and Postgres Integration

This is an ETL data pipeline project utilizing Apache Spark. It combines columns from multiple CSV files, detects failures during processing, and adds a "failure status" column. The pipeline efficiently handles large datasets and securely integrates the transformed data into a Postgres database. It empowers analysis and decision-making based on the processed data.

Customer Churn Analysis Using SQL

Conducted an in-depth analysis on customer churn data for an E-commerce company using SQL. Delved into churn patterns, customer behaviors, and factors influencing customer attrition. This thorough investigation revealed valuable insights into the underlying reasons why customers choose to discontinue their relationship with the company. Drawing from these findings, actionable recommendations were provided to enhance customer retention, fine-tune business strategies, and foster overall growth.

COVID-19 in Nigeria: Exploratory Data Analysis Using Python

Performed exploratory data analysis on the COVID-19 situation in Nigeria using python. Utilized daily data on confirmed, death, and recovered cases from February 2020 to February 2022 across all 36 states. The objective was to uncover meaningful trends and patterns in the spread of the virus. This project shed light on the states in Nigeria most severely impacted by the pandemic, enabling the efficient allocation of resources to those states.

COVID-19 Worldwide Vaccination Data Analysis Using SQL

Analyzed COVID-19 data using SQL to provide meaningful insights into the impact of the virus on different countries and continents, assess the effectiveness of vaccination efforts, and understand the severity of the pandemic over time. The findings from this project contributed to informed decision-making, resource allocation, and the development of effective public health strategies. Some SQL techniques used in this project include joins, CTEs, temp tables, window functions, views, and aggregate functions to derive insights from the data

E-commerce Sales Data Analysis Using Python

Analyzed multiple tables, including customers, orders, geolocation, payments, product, and reviews using Python to uncover patterns and trends in product sales data. Through this analysis, valuable insights were gained to enhance revenue generation, increase profitability, and streamline operational efficiency. Furthermore, a strategic cost-saving approach was devised to mitigate the issue of canceled orders, resulting in substantial savings of over R$250,000 for the company. Power BI was used to visualize the findings.

Data Cleaning of FIFA 21 Dataset Using Python

Cleaned a FIFA 21 dataset using Python programming by identifying and resolving missing values, errors, and inconsistencies. This resulted in a high-quality and analysis-ready dataset that was used to gain valuable insights into the world of football. The dataset was cleaned to ensure data accuracy and consistency, which is essential for any data-driven analysis. This project showcased my expertise in data cleaning, using Python programming, and demonstrated my ability to transform raw data into actionable insights.
